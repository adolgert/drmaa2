DRMAA2 for Python
=================

DRMAA2 is the second version of the Distributed Resource
Management API. This Python module lets you use DRMAA2
to control programs on Univa Grid Engine (UGE), SGE, and
other batch queue systems that supply dynamic libraries
for DRMAA2.
